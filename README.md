# Cosmic Flow - Gravity Sandbox

A healing gravity sandbox running in your browser.
Experience the beauty of physics with N-body simulations, soft light trails, and the mesmerizing evolution of stars into black holes.

## Features

- **N-body Gravity Simulation**: All stars interact with each other.
- **Cosmic Aesthetics**: Deep blue space, glowing particles, and beautiful motion trails.
- **Black Hole Evolution**: massive stars collapse into black holes, consuming everything nearby.
- **Interactive**:
    - **Drag**: Launch new stars.
    - **Right-Click Drag**: Pan the view.
    - **Wheel**: Zoom in/out.

## How to Run

Simply open `gravity_sandbox.html` in any modern web browser.

## Technologies

- HTML5 Canvas
- Vanilla JavaScript
- No external libraries

## Controls

| Action | Effect |
| :--- | :--- |
| **Left Click + Drag** | Create and launch a star. Longer drag = higher velocity. Hold longer = more mass. |
| **Right Click + Drag** | Pan the camera. |
| **Mouse Wheel** | Zoom in and out. |
| **Reset Button** | Clear the universe. |
